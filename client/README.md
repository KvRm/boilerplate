### Запуск локально

```bash
npm install
npm dev
```

После запуска приложние откроется по ссылке http://localhost:3333/

### Сборка приложения

```bash
npm install
npm build
```

### Предпросмотр собранного приложения

```bash
npm preview
```

http://localhost:4173/
